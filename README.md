# SDLOADER-GX.gm9 
Method for installing CIAs on Nintendo 3DS using GodMode9.
- Revision 2026
- Language support: ESP, ENG, FRE, POR, ITA, GER
- Added the color Pink, the signature color of this fork.
- Readjusted certain elements, such as SD or CTRNAND booting in the Luma menu and option alignment, among other tweaks.
- Readjusted parts of the Rosalina menu, such as the sorting of screen filters, among other tweaks.
- Added a link to the technical support Discord for error scenarios: Arm9, Arm11, or Errdisp.
- These adjustments have also been applied to the English version (excluding the translation itself).
- Can be viewed and edited using Visual Studio Code.
- #TeAmoKelonio
- Video Guide: https://www.youtube.com/watch?v=bxgEtnDPr8s

# SDLOADER-GX.gm9 
Método de Instalación de cias en Nintendo 3DS mediante Godmode9. 
- Revisión 2026
- Soporte idiomático: ESP, ENG, FRE, POR, ITA, GER
- Se agregó el color Rosa, siendo el emblema de este fork.
- Se reajustaron algunas partes, como el Arranque SD o Ctrnand en el panel de Luma; o la alineación de sus opciones, entre otros ajustes.
- Se reajustaron algunas partes en el menú de Rosalina, como la ordenación de los filtros de Pantalla, entre otros ajustes.
- Se incorporó el enlace de ayuda Técnica Discord en los escenarios de error: Arm9, Arm11 o Errdisp.
- Estos ajustes también se han propoRcionado en la versión Inglesa, a excepción de la traducción.  
- Puede visualizar y editar desde Visual Studio Code.
- #TeAmoKelonio
- Vídeo Guía:  https://www.youtube.com/watch?v=bxgEtnDPr8s
